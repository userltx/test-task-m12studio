# Тестовое задание m12studio

Репозиторий с решенным тестовым, который соответствует условиям, указанным в Figma по [ссылке](https://www.figma.com/design/EcuMBowopesHHrrUng7crf/Тестовое-задание--frontend-?node-id=0-1&p=f&t=Mp3xhKVZr5QYSanc-0).

## Уточнение

Текст реализован как один абзац без разбиения на строки, согласно требованиям задания. В таком формате браузер автоматически выполняет переносы текста, и управлять тем, на какой строке окажется то или иное слово, технически невозможно. Поэтому, например, слово «ПОД» не может быть зафиксировано на второй строке, если оно помещается на первой.
